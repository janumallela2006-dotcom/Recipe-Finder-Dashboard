# Recipe-Finder-Dashboard
# INTER ID - CITS4013
The Recipe Finder Dashboard is a responsive frontend web application developed using HTML, CSS, and JavaScript. It allows users to browse a variety of recipes, search for recipes by name, and view recipe details through an intuitive and visually appealing interface. The application displays recipe cards with images, categories, and cooking times, along with a View Recipe button that simulates viewing detailed recipe information. Its responsive design ensures a seamless experience across desktop and mobile devices.

Key Features
 Responsive recipe dashboard
 Search recipes by name
 Display recipe cards with images
 Recipe category information
 Cooking time display
 View Recipe functionality
 Modern and attractive user interface
 Mobile-friendly responsive design
